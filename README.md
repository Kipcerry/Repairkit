This script adds a realistic repair kit and tyre kit system with item durability support.

Players can repair damaged vehicles using a repair kit, with repair time and effectiveness fully configurable. Each use reduces the itemâ€™s durability, meaning repair kits and tyres are consumable over time instead of single-use items.

Features

Vehicle repair system using repair kits

Item durability support (repair kits & tyres)
